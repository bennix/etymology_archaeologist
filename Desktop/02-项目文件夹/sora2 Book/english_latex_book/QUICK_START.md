## 🔄 重新编译 / Recompile

如果需要重新编译所有PDF：

```bash
./compile_all.sh
```

这将：
1. 重新编译主书 (main.pdf)
2. 生成所有9个独立章节PDF
3. 生成附录合集PDF
4. 将所有独立PDF放入 `build/individual_pdfs/` 目录
