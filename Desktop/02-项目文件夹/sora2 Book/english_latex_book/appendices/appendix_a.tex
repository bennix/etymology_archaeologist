\chapter{Prompt Syntax Reference}
\label{app:prompt-syntax}

This appendix provides a comprehensive reference for Sora 2 prompt syntax, organized by category for quick lookup during creative work.

\section{Basic Structure}
\index{prompt structure}

\subsection{Essential Components}
Every effective Sora 2 prompt should include:

\begin{enumerate}
\item \textbf{Subject}: Who or what is the main focus
\item \textbf{Action}: What is happening
\item \textbf{Setting}: Where the action takes place
\item \textbf{Style}: Visual and emotional tone
\item \textbf{Technical}: Camera and lighting specifications
\end{enumerate}

\textbf{Template:}
\begin{quote}
[Subject] [Action] in [Setting], [Style], [Camera movement], [Lighting], [Duration]
\end{quote}

\textbf{Example:}
\begin{quote}
A young woman walking through a misty forest, cinematic style, slow tracking shot, golden hour lighting, 8 seconds
\end{quote}

\subsection{Character Types}

\item \textbf{Gender}: man, woman, person, figure
\item \textbf{Profession}: doctor, artist, teacher, chef, musician
\item \textbf{Appearance}: tall, short, athletic, elegant, casual
\end{itemize}

\subsection{Objects and Elements}
\begin{itemize}
\item \textbf{Natural}: tree, flower, mountain, ocean, cloud
\item \textbf{Architectural}: building, bridge, window, door, stairs
\item \textbf{Vehicles}: car, bicycle, train, airplane, boat
\item \textbf{Abstract}: light, shadow, reflection, texture, pattern
\end{itemize}

\section{Action Verbs}
\index{action verbs}

\subsection{Movement Types}
\begin{itemize}
\item \textbf{Gentle}: walking, floating, drifting, swaying, gliding
\item \textbf{Dynamic}: running, jumping, dancing, spinning, flying
\item \textbf{Emotional}: crying, laughing, embracing, contemplating, celebrating
\item \textbf{Interactive}: touching, reaching, pointing, looking, listening
\end{itemize}

\subsection{Temporal Modifiers}
\begin{itemize}
\item \textbf{Speed}: slowly, quickly, gradually, suddenly, smoothly
\item \textbf{Rhythm}: rhythmically, sporadically, continuously, intermittently
\item \textbf{Progression}: beginning, continuing, ending, transitioning
\end{itemize}

\subsection{Cinematic Styles}

\section{Camera Movements}
\index{camera movements}

\subsection{Basic Movements}
\begin{itemize}
\item \textbf{Static}: fixed shot, still frame, stationary camera
\item \textbf{Pan}: left to right, right to left, horizontal sweep
\item \textbf{Tilt}: up and down, vertical movement
\item \textbf{Zoom}: zoom in, zoom out, slow zoom, fast zoom
\end{itemize}

\subsection{Advanced Movements}
\begin{itemize}
\item \textbf{Tracking}: following subject, parallel movement, dolly shot
\item \textbf{Crane}: high to low, sweeping overhead, bird's eye view
\item \textbf{Handheld}: shaky, documentary style, intimate feel
\item \textbf{Steadicam}: smooth movement, flowing motion, professional
\end{itemize}

\section{Lighting Conditions}
\index{lighting conditions}

\subsection{Natural Lighting}
\begin{itemize}
\item \textbf{Golden Hour}: warm, soft, magical, cinematic
\item \textbf{Blue Hour}: cool, mysterious, twilight, atmospheric
\item \textbf{Harsh Sun}: bright, high contrast, dramatic shadows
\item \textbf{Overcast}: soft, even, diffused, natural
\end{itemize}

\subsection{Artificial Lighting}
\begin{itemize}
\item \textbf{Studio}: controlled, professional, clean, bright
\item \textbf{Neon}: colorful, urban, night, electric
\item \textbf{Candlelight}: warm, intimate, flickering, romantic
\item \textbf{Spotlight}: dramatic, focused, theatrical, intense
\end{itemize}

\section{Technical Parameters}
\index{technical parameters}

\subsection{Duration Control}
\begin{itemize}
\item Short clips: 2-4 seconds for quick actions
\item Medium clips: 5-8 seconds for scenes with movement
\item Long clips: 10-15 seconds for complex sequences
\item Extended clips: 20+ seconds for detailed storytelling
\end{itemize}

\subsection{Quality Settings}
\begin{itemize}
\item \textbf{Resolution}: 1080p, 4K, high definition, ultra HD
\item \textbf{Frame Rate}: 24fps cinematic, 30fps standard, 60fps smooth
\item \textbf{Aspect Ratio}: 16:9 widescreen, 4:3 classic, 21:9 cinematic
\item \textbf{Quality}: high quality, professional grade, broadcast ready
\end{itemize}

This reference guide provides the essential vocabulary for creating effective Sora 2 prompts. Combine elements from different categories to achieve specific creative goals.
